# Agent 产品全景：先看清整片森林，再挑树

在开始逐个介绍具体产品之前，我想先带你把整片"Agent 森林"扫一遍。因为这两年冒出来的 Agent 实在太多了，名字一个比一个花哨，很容易看着看着就晕——到底哪个是官方的、哪个是开源的、哪个能写代码、哪个能管生活？如果没有一张整体的地图，你很容易在细节里迷路。所以这一篇不谈安装命令，只谈"格局"：市面上这些 Agent 大致分成几类、各自是什么脾气、强在哪、又适合什么样的人。

## 这些产品其实只有四种"性格"

我自己在用了一圈之后的体会是，市面上看起来眼花缭乱的 Agent，骨子里其实就四种形态，理解了形态，你对一个陌生产品的判断基本就八九不离十了。

第一种是**官方出品的终端编程 Agent**，代表就是 Anthropic 的 Claude Code 和 OpenAI 的 Codex。它们跑在命令行里，直接扎进你的代码库改文件、跑测试、提交 git，是目前综合能力的天花板。它们的共同气质是"贴近真实工程"——不是给你一段建议让你自己抄，而是把活干完再向你汇报。代价是你得习惯终端，对纯新手不算友好。

第二种是**编辑器里的 Agent**，要么是 VS Code 插件（比如开源的 Cline），要么干脆是一整个"AI 原生 IDE"（比如字节的 Trae）。它们最大的好处是可视化、上手快，改动能一眼看到 diff，特别适合不想碰命令行、喜欢点点点的人。国内产品大多集中在这一形态，中文体验也打磨得最好。

第三种是**自托管的个人助理 Agent**，这两年最出圈的就是 OpenClaw 和 Nous 的 Hermes。它们跳出了"只写代码"的框，能连你的文件、浏览器、甚至微信 WhatsApp 这类聊天软件，7×24 帮你自动化处理各种杂事，更像是"装在你自己电脑上的贾维斯"。强是真的强，但因为它能碰你的真实世界，安全责任也实打实地压在你自己肩上。

第四种我把它叫做**自主任务型 Agent**，典型是开源的 OpenHands，你给它一个高层目标，它自己拆解、自己跑完，最接近大家想象中的"数字员工"。自动化程度最高，但也最需要你盯着，成本和不可控性都更高。

把这四种形态记在脑子里，后面每篇具体产品你都能迅速对号入座。

## 三大阵营：官方、开源、国产

除了按形态分，还有一个更接地气的分法，就是看它出自谁手——这直接决定了你的钱包和网络会不会舒服。

**官方阵营**是 Claude Code 和 Codex，背靠 Anthropic 和 OpenAI 自家最强的模型，能力最能打，但要付费、且对国内网络不太友好。值得一提的是，Codex 在 2026 年 7 月已经并进了 ChatGPT，不再单独订阅，这个变化后面那篇会细讲。

**开源阵营**热闹非凡，OpenHands、Cline、Aider、opencode、Goose 各有拥趸，共同的好处是免费、可以自带 API Key 甚至跑本地模型、一切尽在掌控。如果你预算敏感又爱折腾，这一挂几乎能满足你大部分需求。而 OpenClaw 和 Hermes 虽然也开源，但它们志不在编程，是奔着"通用个人助理"去的，属于开源阵营里另一个物种。

**国产阵营**是我特别想推荐给国内朋友的一档。通义灵码、Qwen Code、Trae、iFlow、CodeBuddy、文心快码、CodeGeeX 这一大串，最大的三个优势就是中文交流自然、免费额度大方、访问没有网络障碍。其中阿里的 Qwen Code 还是开源终端形态，基本可以理解为"国产版 Claude Code"，是里面我最看好的一个。

## 一张表先记个大概

文字讲完格局，我还是放一张表让你有个整体印象。真正做决定时可以直接翻到最后那篇《横向对比与选型》，那里有更细的场景推荐；这里先混个脸熟就好。

| 产品 | 形态 | 阵营 | 一句话印象 |
|---|---|---|---|
| Claude Code | 终端 CLI | Anthropic 官方 | 综合最强，工程党首选 |
| OpenAI Codex | ChatGPT 内 + CLI | OpenAI 官方 | 已并入 ChatGPT，生态用户顺手 |
| OpenClaw | 自托管助理 | 开源 | 爆火的全能个人助理 |
| Hermes | 自托管助理 | Nous Research | 会自己成长的私有 Agent |
| OpenHands | 自主平台 | 开源 | 开源版"数字员工" |
| Cline | VS Code 插件 | 开源 | 编辑器里最稳的可视化选择 |
| Aider / opencode / Goose | 终端 CLI | 开源 | 轻量、不锁厂商、可玩性高 |
| Qwen Code | 终端 CLI | 阿里（开源） | 国产版 Claude Code |
| Trae / 通义灵码等 | IDE | 国内厂商 | 中文友好、开箱即用 |

## 挑之前，先想清楚三件事

聊了这么多产品，其实真正帮你做决定的不是"哪个最强"，而是三个关于你自己的问题。

先问自己**你是终端党还是编辑器党**。这一个问题就能把选择范围砍掉一半——愿意用命令行，Claude Code、Codex、Qwen Code 的世界向你敞开；只想在编辑器里点，那就去看 Trae、通义灵码、Cline。形态选对了，体验就对了一大半。

再问**你要它干的是编程还是生活**。如果你只是想写代码、做科研、跑数据，那就在编程 Agent 里挑，别去碰那些个人助理，越专注越好用；但如果你的野心是让 AI 帮你管日程、理邮件、自动化各种琐事，那 OpenClaw 这类才是你的菜。

最后一个问题最容易被忽略，却最重要：**你愿意为安全负多少责任**。官方和国产的托管产品，安全边界基本由厂商兜着；而 OpenClaw、Hermes、OpenHands 这些自托管的家伙，你给它的每一份权限、每一个 API Key、每一个第三方 Skill，都是实打实的攻击面。它们很强，但用之前请务必接受一个前提——自托管就等于自己扛安全。这也是为什么我在后面每一篇里都会专门留一节讲注意事项。

想清楚这三件事，你再往下翻，读到哪个产品都会心里有数。那么，就从公认最强的 Claude Code 开始吧。
